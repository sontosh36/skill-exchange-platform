# SkillSwap - Skill Exchange Platform

**Live Website**: **

---
## Purposes
### 1. Learn Anything, AnyTime from Anyone
- Users can find local people who teach skills -from guitar or coding or baking.
- Learning becomes flexiable and accessible, not limited to schools or instutions.
- You can pick your schedule, your style of learning.
---
### 2. Community based & Localized Learning
- The platform focuses on your local area, so user can meet and learn face to face.
- Bulids a stronger community connection -- people help each other grow.
- Great for praticing skills that needed in-person interaction.

---
### 3. Opportunity to Earn r trade Skills
- Individuals can offer their skills as services (free or paid).
- You can also exchange skills - e.g. teach someone guitar in return for learning cooking.
- It encourages a sharing economy - knowledge as a currency.

---
### 4. Reviews and Ratings Bulid Trust
- The Rating system help learners find reliable teacher.
- Providers get reputation and credibility , leading to more opportunities.

---
## Key Features
**Email & Google Authentication**

Users can register or log in using email/password and Google account. Password validation ensure security.

**Protected Route**

Pages like "Profile" and "skillDetails" are secured and accessible only after login.

**Responsive Design**

Full Responsive for mobile , tablet, Desktoo device.

**Toast**

Users action like login, logout show real time feedback with toast or alert.

## Technology & NPM Used
- **React** - Frontend UI design
- **React-Router** - Routing
- **Firebase** - Authentication and Hosting
- **Axios** - For API Communication
- **Tailwindcss** - Utility-first CSS framework
- **DaisyUI** - Component Library for Tailwindcss
- **React-Toastify** - Non Blocking Toast Notification
- **React-icons** - Icon Library
- **SwiperJS** - Swiper Slider for UI element
